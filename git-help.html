Памятка по работе с git в терминале, которы сначала надо{" "} установить

Абсолютно точно это не хороший и подробный разбор возможностей гита. Это просто памятка для работы в терминале с гитом для таких же ничего не понимающий как и я, чтобы хоть что-то работало

git init - просто инициализируем гит в проекте

git remote add origin ССЫЛКА_НА_ПРОЕКТ или git remote set-url origin ССЫЛКА_НА_ПРОЕКТ {" "} - связка гита с проектом

git status - узнать что вообще происходит

git add .ССЫЛКА_НА_ФАЙЛ - добавить на отпраку изменения

git commit -m 'Здесь ваше сообщение' - добавить коммит в текущую ветку с сообщением(-m) в кавычках

git push - отпарвить измениения в гит

git checkout -b НАЗВАНИЕ_ВЕТКИ - создать и перейти в новую ветку

git log --pretty=oneline - вывод лога в определённом виде. Здесь в одну строку строку

git log --since='1 hour' - вывод лога за какое-то время

git log -p -СКОЛЬКО_КОММИТОВ - вывод определённого числа коммитов

git reset HEAD НАЗВАНИЕ_ФАЙЛА - удалить файл из состояния staged. Точка вместо названия файла если надо удалить всё

git commit --amend - добавать изменения к последнему коммиту. Нажать i чтобы изменить сообщение, esc чтобы выйти из режима редактирования и :!x чтобы всё это закончить

git restore НАЗВАНИЕ_ФАЙЛА - откатить изменения в файле, который ещё не был добавлен в staged. Точка вместо названия файла если надо откатить все изменения

git revert ХЭШ_КОММИТА - откатить изменения этого коммита. Нажать i чтобы изменить сообщение, esc чтобы выйти из режима редактирования и :!x чтобы всё это закончить. Будет создано новый коммит с откато измениени и сообщением типа revert that's commit message

git checkout ХЭШ_КОММИТА - посмотреть что было в том коммите

git branch - посмотреть все ветви

git branch НАЗВАНИЕ_ВЕТВИ - создать новую ветку, но не переключиться на неё

git checkout НАЗВАНИЕ_ВЕТВИ - переключиться на ветвь

git log --oneline --decorate --graph --all - красиво вывести всё с ветками

git merge НАЗВАНИЕ_ВЕТВИ - слить с текущей ветвю ветку НАЗВАНИЕ_ВЕТВИ

git branch -d НАЗВАНИЕ_ВЕТВИ - удалить ветку, нельзя сделать находясь на ней
